
Week 1: How the Internet Talks

My first model of how the internet works

What I learned from TryHackMe "What is Networking?"

1. Networking = devices talking to each other using rules called protocols

2. The Internet = giant network of networks connected by ISPs

3. IP addresses = home address for devices. Example: 8.8.8.8 is Google DNS

4. Ping uses ICMP protocol to test if a device is reachable

5. MAC addresses = physical hardware ID burned into each device

My model:

When I type a website, my request goes:

Laptop Router ISP DNS finds the IP ← Data travels via TCP/IP ← Website server sends the page back.
